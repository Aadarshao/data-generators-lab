# Attendance Spark Pipeline

This project demonstrates how to use the `employee_attendance` scenario
with a Spark / PySpark pipeline.

- `notebooks/01_data_design.ipynb` – design of the attendance dataset
- `notebooks/02_generate_attendance_data.ipynb` – generation & export
- `configs/attendance_scenario_local.yml` – parameters for local runs
- `scripts/run_generate_attendance.ps1` – helper script
